# SecureCryptoX

SecureCryptoX is a secure cryptocurrency wallet management system built using **Flask** and **Microsoft SQL Server**.

## Features
- User Registration & Login
- Deposit & Withdraw Money
- Buy & Sell BTC
- BTC Transfer Between Users
- Wallet Management
- Transaction History

## Security Features
- Password Hashing
- Role-Based Access Control (RBAC)
- Row-Level Security (RLS)
- Dynamic Data Masking (DDM)
- AES-256 Encryption
- Audit Logging
- Suspicious Activity Monitoring
- CSRF Protection
- SQL Injection Prevention

## Technologies Used
- Python Flask
- Microsoft SQL Server
- PyODBC
- HTML, CSS, JavaScript

## Run Project

Install dependencies:

```bash
pip install -r requirements.txt
```

Run application:

```bash
python app.py
```

## Roles
- **Admin** → Full access  
- **Manager** → Monitor suspicious activities  
- **User** → Wallet operations only
